Color Contrast Checker

Free WCAG 2.2 contrast checker — drop-in Web Component, zero dependencies.

How to use it

Add the script, then use the element anywhere — plain HTML, React, Vue, Svelte or Astro:

<script src="contrast-checker.js"></script>
<contrast-checker foreground="#1a1a1a" background="#ffffff"></contrast-checker>

Why contrast matters

WCAG 2.2 requires a contrast ratio of at least 4.5:1 for normal text and 3:1 for large text (AA). Low-contrast text is the single most common accessibility failure on small-business websites.
